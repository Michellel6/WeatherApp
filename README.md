
🌦️ Weather Dashboard
A modern, responsive weather application built using HTML, CSS, and JavaScript. The application provides real-time weather information, a 5-day forecast, live clocks, and an interactive user experience powered by the OpenWeather API.

📖 Overview

Weather Dashboard allows users to search for any city around the world and instantly view current weather conditions along with a 5-day weather forecast. It also includes location-based weather, temperature charts, search history, and favorite cities for quick access.

This project was created to strengthen my front-end web development skills by working with APIs, asynchronous JavaScript, local storage, and responsive user interface design.

✨ Features Include:

🔍 Search weather by city name
📍 Get weather for your current location
🌡️ Current temperature and "Feels Like" temperature
☁️ Weather condition and icon
💧 Humidity
💨 Wind speed
👀 Visibility
🌡️ Atmospheric pressure
🌅 Sunrise and sunset times
📅 5-Day Weather Forecast
📈 Interactive temperature chart
🕒 Live local clock
🌍 Live clock for the searched city
⭐ Save favourite cities
🎨 Dynamic weather backgrounds
📱 Responsive design for desktop and mobile devices
